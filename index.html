Linux-Robotic-Arm

The following is a software architecture and project that depicts a Linux robotic arm application that works in userspace to control a scalable N joint robotic arm for small embedded systems.

Each joint positioning is controlled on a separate thread so it is useful in multi-core environments that have good GPIO and PWM capabilities such as the Intel Edison boards.
